# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in cummand, please report it privately.

Open a [security advisory](https://github.com/divyanshudhruv/cummand/security/advisories/new)
or email the maintainer directly.

Do not open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.4.x   | Yes       |
| < 0.4   | No        |
